## Burp suite basics
Simply put, Burp Suite captures and enables manipulation of all the HTTP/HTTPS traffic between a browser and a web server.

There are 3 versions of Burp Suite:
- Community Edition (Free)
- Professional Edition (Paid)
- Enterprise Edition (Paid - for continuous scanning)

## Community Edition features
- Proxy - enables interception and modification of requests and responses while interacting with web applications,
- Repeater - llows for capturing, modifying, and resending the same request multiple times,
- Intruder - automates customized attacks, such as brute-forcing login credentials or fuzzing input fields,
- Decoder - encodes and decodes data in various formats (e.g., URL encoding, Base64, Hex),
- Comparer - compares two pieces of data to identify differences,
- Sequencer - analyzes the randomness of tokens or session IDs.
